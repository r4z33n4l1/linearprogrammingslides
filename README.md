# Linear Programming Presentation

This workspace now includes a `manim` + `manim-slides` presentation for a MAT392 talk on linear programming.

## Files

- `slides.py`: Main presentation source
- `requirements.txt`: Python dependencies for the slide deck
- `media/`: Auto-generated by Manim after rendering

## Setup

Create and activate a virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

Install the presentation dependencies:

```bash
pip install -r requirements.txt
```

Verify the tools are available:

```bash
manim --version
manim-slides --version
```

## Render And Present

Render the slides in low quality for quick iteration:

```bash
manim-slides render slides.py LPPresentation -ql
```

Open the interactive presentation:

```bash
manim-slides LPPresentation
```

Export to HTML if needed:

```bash
manim-slides convert LPPresentation presentation.html -ccontrols=true
```

## Presentation Outline

The deck covers:

1. Title and roadmap
2. Factory analogy and motivation
3. Convexity and convex vs. non-convex sets
4. Linear programming definition and feasible regions
5. Building a feasible polytope from constraints
6. Vertices and the fundamental theorem of linear programming
7. A worked example with objective values at each vertex
8. Historical context and closing questions

## Controls

- Right arrow or click: next slide
- Left arrow: previous slide
- `Q`: quit
